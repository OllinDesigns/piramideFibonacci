Como hacer el ciclo Fibonacci en JavaScript


Lo primero que debemos hacer es definir nuestras variables, 3 en este caso


let primerNumero = 1;
let segundoNumero = 1;
let miArray = [primerNumero, segundoNumero];



Después definimos un bucle for con los siguientes parámetros


for(let i = 0; i <= 50; i++){

let nuevoNumero = primerNumero + segundoNumero;
miArray.push(nuevoNumero);
primerNumero = segundoNumero;
segundoNumero = nuevoNumero;
}


Le asigno un argumento de i <= a 50, para no hacerlo muy largo



Este código JavaScript ejecutará la serie Fibonacci. Ahora procederemos a imprimirlo


Lo imprimiremos en forma de pirámide, pues de lo contario haría una línea muy larga con todas estas cifras. De esta manera también le damos un aspecto más agradable y más fácil de leer. Aquí insertamos un salto /br cada vez que se cumple la condición del bucle. Para esto tenemos que utilizar el siguiente bucle for


for(x of miArray){

document.write(x + "< br >");

}
